## Team Members

## Project Overview
A full-stack restaurant order app built with React and Express. Users can view existing orders and create new orders through the frontend, which communicates with the backend API.

## Features
- View restaurant orders
- Create a new order
- Frontend connected to backend API
- Fully containerized with Docker Compose
- Automated backend tests

## Tech Stack
- Frontend: React + Vite
- Backend: Express
- Containers: Docker + Docker Compose

## How to Run
```bash
docker compose up --build
